Understanding the stack

Sebelum setup, penting untuk paham 3 komponen utama.

⌨️
Claude Code
Agentic coding tool dari Anthropic yang hidup di terminal. Bisa build fitur, debug, navigasi codebase, dan otomasi task — sekarang juga bisa bicara langsung ke Figma.
AI Agent
🔌
MCP (Model Context Protocol)
Standar terbuka yang memungkinkan AI tools terhubung ke berbagai data source dan aplikasi. Ibaratkan sebagai "USB universal" — satu protokol, banyak koneksi.
Protocol
🎨
Figma MCP Server
Server yang menjembatani Figma dengan AI tools. Bisa membaca komponen, variabel, style, dan layout — lalu mengirim UI dari browser ke canvas sebagai editable layers.
Bridge

Apa itu MCP?
Model Context Protocol — standar terbuka yang memungkinkan AI tools terhubung ke aplikasi eksternal. Seperti USB tapi untuk AI.
Kenapa butuh MCP?
Tanpa MCP, Claude Code tidak bisa membaca desain Figma atau mengirim UI ke canvas. MCP menjadi "jembatan" antara keduanya.
Apa yang dikirim MCP?
Bukan screenshot — MCP mengirim komponen, variabel, style, dan layout structure. Claude memahami design system secara semantik.

Step 1 of 2

Install Claude Code dan plugin Figma.

1/3

Install Claude Code

Jalankan satu command di terminal untuk menginstall Claude Code.

Terminal — zsh
ngereige@MXHWL27PQ9 ~ % curl -fsSL https://claude.ai/install.sh | bash Setting up Claude Code...   ✓ Claude Code successfully installed!   Version: 2.1.50   Next: Run claude --help to get started   ⚠ Setup notes: • Native installation exists but ~/.local/bin is not in your PATH. Run:   echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc   ☑ Installation complete! ngereige@MXHWL27PQ9 ~ % █
💡
Pastikan Node.js sudah terinstall. Jika PATH belum disetting, jalankan perintah echo yang ditampilkan.
2/3

Launch Claude Code

Ketik "claude" di terminal untuk memulai. macOS mungkin minta izin akses.

Terminal — zsh
ngereige@MXHWL27PQ9 ~ % claude

3/3

Install Figma Plugin

Install plugin resmi Figma langsung dari marketplace Claude Code.

Claude Code
/plugin install figma@claude-plugin-directory Plugin 'figma@claude-plugins-official' is already installed. Use '/plugin' to manage existing plugins.   > /plugin █
💡
Alternatif: gunakan command claude mcp add --transport http figma https://mcp.figma.com/mcp di terminal biasa.
✓ Install done — next: Connect MCP

Step 2 of 2

Hubungkan Claude Code ke Figma via MCP authentication.

1/3

Enable & Authenticate

Ketik /plugin, pilih figma-remote-mcp, lalu pilih Authenticate.

Claude Code
/plugin   Plugins Discover Installed Marketplaces   Figma-remote-mcp MCP Server   Status: ○ disabled Auth: ✕ not authenticated URL: https://mcp.figma.com/mcp Config: /Users/ngereige/.claude.json   > 1. Enable 2. Authenticate   ↑↓ to navigate · Enter to select · Esc to back
💡
Pilih "Enable" dulu, lalu "Authenticate". Browser akan terbuka otomatis.
2/3

Allow Access di Figma

Klik "Agree & Allow Access" pada halaman OAuth Figma.

Claude Code would like to access your account and be able to:

Switch accounts

Authentication Successful

💡
Pastikan kamu login ke akun Figma yang benar sebelum klik Allow Access.
3/3

Verify Connection

Kembali ke Claude Code, ketik /plugin untuk memastikan status connected.

Claude Code
/plugin Authentication successful. Connected to figma-remote-mcp.   /plugin Plugins Discover Installed Marketplaces   ○ Search...   Local > figma-remote-mcp MCP · ✓ connected   claudeai claude.ai Figma MCP · ✓ connected   type to search · Space to toggle · Enter to details
💡
Jika masih disconnected, restart Claude Code dan ulangi proses authenticate.
✓ Setup complete!
